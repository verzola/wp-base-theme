# Wordpress Base Theme

Webpack-based base theme for Wordpress

## Features
- Organized
- Scalable
- Fast

## Tools
- Webpack
- SASS
- Autoprefixer
- Babel
- Browsersync
- ESLint
- Stylelint
- Prettier
- Husky
- Lint-staged
- Prettier
- Normalize.css
- Hamburgers

## Setup
```sh
# enter in themes directory in your wordpress installation
$ cd wp-content/themes/

# clone base theme
$ git clone https://github.com/verzola/wp-base-theme.git

# enter in base theme directory
$ cd wp-base-theme

# install dependencies
$ npm install

# run browsersync development server
$ npm run dev

# compile assets for production
$ npm run build
```
